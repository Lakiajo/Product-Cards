# Product Cards are dope

This is my Product Card Page. It allows you to view all available and unavailable cupcakes. 

## Getting Started
```
$ git clone git@github.com:Lakiajo/Product-Cards.git
```

### Prerequisites
```
$ npm install -g http-server
```
